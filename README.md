README.md🧬 qPCR PrimerBatch Designer (High-GC Specialized)基于热力学优化的 qPCR 引物批量设计与可视化系统📖 项目简介qPCR PrimerBatch Designer 是一款专为复杂基因组研究者开发的轻量化引物设计工具。特别针对以结核分枝杆菌为代表的 高 GC 含量基因组（GC > 60%）进行了算法强化，解决了通用引物设计软件在处理此类高难度模板时匹配率低、特异性差的痛点。本项目采用纯前端架构，无需安装任何后台环境，即可在浏览器中实现从序列输入到位置可视化的完整工作流。✨ 核心优势高 GC 适配算法：内置宽 $T_m$ 搜索逻辑（支持 55°C - 75°C 区间），在极端序列中寻找热力学平衡点。批量并行处理：支持同时输入数十条 FASTA 序列，一次性完成全基因组多靶点引物筛选。动态序列地图：自动渲染引物在模板序列上的物理位置，使用颜色高亮区分 Forward 和 Reverse 绑定区。3' 端稳定性校验：严格过滤 3' 末端 GC 含量过高的候选，有效降低非特异性扩增和引物二聚体风险。科研数据导出：结果支持一键复制，格式完美兼容 Excel 等科研记录表格。🚀 快速启动获取项目:Bashgit clone https://github.com/wwp643/qPCR-Primer-Designer.git
使用: 直接用浏览器打开 index.html 即可使用。工作流程:粘贴单条或多条 FASTA 格式序列。根据实验需求调整 $T_m$ 范围和产物长度。点击 "Run Design" 实时获取最优引物组合。🧪 实验优化建议对于高 GC 样本（如分枝杆菌、链霉菌等），建议在实验中参考以下配置：热启动策略：使用 Hot-Start DNA 聚合酶以减少初期非特异性扩增。助溶剂：在体系中加入适量 DMSO 或 Betaine。程序优化：若背景较深，可尝试使用 Touch-down PCR 程序。🛠️ 贡献与反馈如果你在实验中遇到任何问题，或对算法有改进建议，欢迎提交 Issue。项目维护者: wwp643
